# Lucky Order 跨境电商出库数据分析平台

> 完整项目说明文档 / 2026-06-01 V2 版本

---

## 一、项目定位

**Lucky Order** 是一个跨境电商 ERP 订单智能分析平台。从马帮 ERP 导出 Excel 后拖入网页，系统自动清洗、聚合、标准化，生成经营驾驶舱、AI 洞察、国家热力图、新品追踪等分析功能。多人注册使用，数据隔离。

---

## 二、技术栈

| 层 | 选型 | 说明 |
|:---|:---|:---|
| 前端框架 | React 19 + Vite 8 | SPA 单页应用 |
| 图表 | ECharts 5 + echarts-for-react | 地图、柱状图、饼图、折线图、散点图 |
| 数据库 + 认证 | Supabase | 免费额度，邮箱注册，RLS 行级安全 |
| 智能分类 | 本地引擎（classifier.js） | 关键词→品类映射，免 API |
| 分析引擎 | 本地计算（analyzer.js） | 基于统计的自动分析，免 API |
| 导入引擎 | xlsx （SheetJS） | 解析马帮 ERP 导出 Excel |
| 导出图片 | html2canvas | 截取看板为 PNG |
| 地图数据 | world-atlas | 本地 GeoJSON，无 CDN 依赖 |
| 部署 | GitHub Pages + Actions CI/CD | 自动构建部署 |
| 域名 | luckyck.cn（CNAME → GitHub Pages） | 海外服务器，无需备案 |

## 四、数据结构（Supabase `orders` 表）

| 字段 | 类型 | 说明 | 来源 |
|:---|:---|:---|:---|
| id | BIGINT (PK) | 自增主键 | 自动 |
| user_id | UUID | 关联 auth.users | 自动 |
| created_at | TIMESTAMPTZ | 创建时间 | 自动 |
| updated_at | TIMESTAMPTZ | 更新时间 | 自动 |
| **order_no** | TEXT | 订单编号 | 马帮「订单编号」 |
| **store_name** | TEXT | 所属运营店铺 | 马帮「店铺名」 |
| **country** | TEXT | 国家（已标准化） | 马帮「国家」→ 标准化（US→美国） |
| **province** | TEXT | 州/省 | 马帮「所属地区（省/州）」 |
| **product_category** | TEXT | 产品品类 | 自动分类（关键词映射） |
| **product_name** | TEXT | 产品名称/SKU描述 | 马帮「订单商品名称」聚合 |
| **product_sku** | TEXT | 库存SKU | 马帮「SKU」聚合 |
| **quantity** | INTEGER | 商品数量 | 马帮「SKU总数量」聚合 |
| **total_amount** | DECIMAL | 总金额 | 马帮可导入（当前默认0） |
| order_status | TEXT | 订单状态 | 全部默认为 completed |
| order_date | DATE | 发货日期 | 马帮「发货时间」取年月日 |

## 五、已实现功能清单

### 📥 数据导入（马帮引擎 V2）

- [x] **直接识别马帮原始导出文件**（无需改格式）
- [x] **空白订单号向下填充**：空白行自动继承上一行订单号
- [x] **多 SKU 自动聚合**：同订单所有 SKU 聚合为一条记录（订单维度统计）
- [x] **国家标准化**：30+ 国家自动标准化（US→美国、UK→英国、DE→德国）
- [x] **日期拆分**：自动提取年-月-日 + 星期几 + 月份字段
- [x] **智能分类**：关键词规则自动推断品类
- [x] **清洗预览**：显示原始行数/合并后订单数/填充数/标准化数
- [x] **支持通用格式**：非马帮格式也能自动识别字段映射

### 📊 经营驾驶舱（V2 首页）

- [x] **核心 KPI**：总订单 / 总销售额 / 总出库量 / 客单价 / 覆盖国家 / 运营店铺
- [x] **多期环比**：每个 KPI 自动显示 **昨日↑↓、上周↑↓、上月↑↓** 百分比
- [x] **AI 经营洞察**：首页显眼位置，自动发现 ⚠️ 异常下降 + 🔥 增长机会
- [x] **销售趋势**：支持 **7天 / 30天 / 90天** 切换
- [x] **异常监控**：下降最快品类 / 店铺 / 国家，红色高亮 + 数据详情
- [x] **全球订单分布**：ECharts 世界地图散点图 + 国家排行 TOP 10
- [x] **国家详情弹窗**：近30天趋势、品类分布、SKU排行、AI 分析

### 📈 详细分析页面

- [x] **品类概览**：出库排行柱状图 + 占比环形图
- [x] **趋势分析**：7/30/90天折线图 + 月趋势
- [x] **热力图**：周几 × 周次 出库热力矩阵
- [x] **店铺分析**：店铺出库排行 + 占比
- [x] **供应商分析**：供应商排名 + 状态分布 + 完成率
- [x] **国家分析**：国家排行 + 州/省分布（偏远地区分析）
- [x] **数据明细**：全部订单表格，支持搜索/筛选

### ✨ 新品追踪

- [x] 添加新品 + 上线日期
- [x] 自动匹配订单出库数据
- [x] 订单数 / 出库量 / 日均出库
- [x] 每日趋势折线图
- [x] 上线前数据预警

### 🧠 AI 洞察（独立页面）

- [x] 本地计算引擎（不限次数、永久免费）
- [x] 整体概况 + 品类分析 + 趋势判断 + 订单健康度 + 运营建议

### 🔧 工具功能

- [x] **智能分类规则管理**：新增/删除/恢复默认

## 七、数据导入流程

```
马帮导出 .xlsx
    ↓
拖入网页
    ↓
XLSX 解析 → 检测马帮格式
    ↓
Step 1: 空白订单号向下填充
Step 2: 国家名标准化（US→美国）
Step 3: 日期提取（年月日/星期/月份）
Step 4: 按订单号聚合（多SKU合并）
    ↓
清洗预览（显示原始行数/合并订单数/填充数/标准化数）
    ↓
确认导入 → 写入 Supabase
    ↓
分析生效（看板/AI/地图自动更新）
```
